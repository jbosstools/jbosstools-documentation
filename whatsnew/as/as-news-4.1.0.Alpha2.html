Increased support for server runtimes

Support has been increased for various server runtime types. First, a bare-minumum of support has been guaranteed for various products which will be based on JBoss AS. This ensures that we should (hopefully) never have a situation where a new product release fully requires a corresponding release of tools to add support. Extended support may be added in future JBossTools versions, but a situation where servers completely fail should now be a thing of the past.

Also, support for previous and current versions of Gate-In have been added. This ensures that Gate-In receives the same level of support that you've come to expect from our other server adapters.
