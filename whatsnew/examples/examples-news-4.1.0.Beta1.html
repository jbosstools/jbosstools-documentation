Project Examples
Opening cheatsheet on import project

Now, the project examples plugin opens the .cheatsheet.xml/cheatsheet.xml when a project is imported.
This feature must be enabled using the Window>Preferences>JBoss Tools>Project Examples preferences page.

Related JIRA : JBIDE-14332
Show in Cheat Sheets view.

JBoss Tools includes a new action in the "Show in" context menu: the Show in Cheat Sheets action.
The action is enabled for the Java, J2EE, JBoss and PDE perspective.

See http://screencast.com/t/gK5JggVU

Related JIRA : JBIDE-14334
New cheatsheet commands

We have added new context sensitive parameterized commands for open file, show type etc. The commands can be used when creating cheatsheets.

See http://screencast.com/t/gK5JggVU

Related JIRA : JBIDE-14333
